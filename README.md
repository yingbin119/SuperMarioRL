# 依赖安装
```bash
uv sync --python 3.8
pip install -r requirements.txt
```
* 注意每次运行'uv sync'都需要重新安装requirements.txt中的依赖。
* 还有pyproject.toml中cuda的版本也要根据你自己的情况调整一下
